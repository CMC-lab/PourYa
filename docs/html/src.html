Fetch and plot daily data for specified metrics.

Parameters:

  • data_type – Type of data to fetch and plot (e.g., ‘daily_activity’).

  • source – Data source, either ‘csv’ or ‘api’.

  • start_date – Start date in ‘YYYY-MM-DD’ format.

  • end_date – End date in ‘YYYY-MM-DD’ format (optional).

  • caption – Caption for the plot (optional).

  • peacewisedDays – Number of days to aggregate for the plot.

  • detect_trends – Whether to detect and plot trends.

  • n_bkps – Number of breakpoints to detect for trends.

  • detect_anomalies – Whether to detect and highlight anomalies.

  • anomaly_threshold – Z-score threshold for anomalies.
